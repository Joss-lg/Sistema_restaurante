2️⃣ Implementación Modular (Clase Reutilizable)

Si quieres usar la clase PlanoEspacialMesas en otras vistas:

<!-- En tu vista Blade -->
<script src="{{ asset('js/plano-espacial.js') }}"></script>

<script>
    // Inicializar la clase
    const plano = new PlanoEspacialMesas({
        apiBase: '{{ route("admin.plano-espacial.api.mesas") }}',
        apiGuardar: '{{ route("admin.plano-espacial.api.guardar") }}',
        apiStore: '{{ route("admin.plano-espacial.api.store") }}',
        apiEliminar: '{{ route("admin.plano-espacial.api.eliminar", ["id" => "ID"]) }}',
        csrfToken: '{{ csrf_token() }}'
    });

    // Iniciar
    plano.init();
</script>

✓ Ventaja: Reutilizable en múltiples páginas y componentes

3️⃣ API de la Clase PlanoEspacialMesas

Métodos Públicos

init()

Inicializa el sistema: carga mesas, cachea elementos, configura eventos

await plano.init();
cargarMesas(zona = '')

Carga mesas desde el servidor, opcionalmente filtradas por zona

await plano.cargarMesas();           // Todas
await plano.cargarMesas('salon');   // Solo salón
await plano.cargarMesas('terraza'); // Solo terraza
activarEdicion()

Activa el modo edición: habilita drag & drop

plano.activarEdicion();
desactivarEdicion()

Desactiva el modo edición: restaura estado original

plano.desactivarEdicion();
guardar()

Guarda el plano actual en la base de datos

await plano.guardar();
crearMesa()

Crea una nueva mesa desde los campos del modal

await plano.crearMesa();
seleccionar(mesa)

Selecciona una mesa y muestra sus propiedades

const mesa = estado.mesasActuales[0];
plano.seleccionar(mesa);
abrirModal()

Abre el modal de creación de mesa

plano.abrirModal();
cerrarModal()

Cierra el modal de creación de mesa

plano.cerrarModal();
notificar(mensaje, tipo)

Muestra una notificación (success, error, info)

plano.notificar('Mesa creada', 'success');
plano.notificar('Error al guardar', 'error');
plano.notificar('Información', 'info');

4️⃣ Ejemplo Completo de Integración

<!-- Incluir la librería -->
<script src="{{ asset('js/plano-espacial.js') }}"></script>

<script>
    document.addEventListener('DOMContentLoaded', async () => {
        // 1. Crear instancia
        const plano = new PlanoEspacialMesas();
        
        // 2. Inicializar
        await plano.init();
        
        // 3. Escuchar eventos personalizados (opcional)
        document.getElementById('btnEditar').addEventListener('click', () => {
            plano.activarEdicion();
            console.log('Modo edición activado');
        });
        
        // 4. Crear mesa programáticamente (ejemplo)
        window.crearMesaProgramada = async () => {
            plano.elementos.inputNumero.value = 'M-NUEVA';
            plano.elementos.inputCapacidad.value = 6;
            await plano.crearMesa();
        };
    });
</script>

6️⃣ Validaciones

Frontend (Validación Rápida)

  • Número no puede estar vacío
  • Capacidad debe ser número válido (1-20)
  • Solo valores permitidos en selects

Backend (Validación Segura)

  • Número única en tabla mesas
  • Capacidad entre 1 y 20
  • Estado en enum permitido
  • Protección CSRF
  • Autenticación requerida

8️⃣ Troubleshooting

Problema: Número duplicado

Solución: Usar número único o verificar en BD: SELECT * FROM mesas WHERE numero = 'M1';

Problema: Drag & drop no funciona

Solución: Asegurar que modo edición esté activo: plano.activarEdicion();

Problema: Mesa no aparece en plano

Solución: Revisar consola (F12) → Network → Check respuesta de POST

Problema: CSRF Token missing

Solución: Agregar en Blade: <meta name="csrf-token" content="{{ csrf_token() }}">

📞 Soporte Técnico

En caso de errores:

  1. Revisar archivos de log: storage/logs/laravel.log
  2. Verificar consola del navegador (F12)
  3. Consultar respuesta de red en Developer Tools
  4. Comprobar permisos en base de datos
  5. Ejecutar php artisan migrate:refresh --seed si es necesario
